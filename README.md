## Installation

```bash
npm install
# or
yarn install
```

## Rodando a aplicação

Primeiro vamos iniciar a FakeAPI e o servidor.

```bash
npm run server
# or
yarn server
```

```bash
npm run start
# or
yarn start
```

Abra no navegador para ver o resultado:
[http://localhost:3000](http://localhost:3000)
FakeAPI: http://localhost:3333/stock & http://localhost:3333/products
